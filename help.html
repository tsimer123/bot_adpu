Добро пожаловать в бот https://t.me/SEK_ADPU_bot Данный бот поддерживает следующие команды: - /start – начало работы с ботом - /help – вызов справки - /coord – по заданным координатам выдает ближайшее опорное оборудование в радиусе 500 метров. Формат команды: /coord 55.706792, 37.625540. Бот принимает координаты в пределах Московской области. - /tel – выводит информацию о sim карте по номеру телефона. Формат команды: /tel 79250012194 - /iccid – выводит информацию по sim карте по номеру sim карты – iccid (нанесенный на sim кату номер, содержащий 17-20 цифр в зависимости от оператора связи). Формат команды: /iccid 89701010050357107028. Примеры команд для разных операторов связи: МТС (20 цифр) - /iccid 89701010050357107028 Мегафон (17 цифр) - /iccid 89701026275366365 Билайн (19 цифр) - /iccid 8970199120901329444 - /imsi – выводит информацию по sim карте по номеру sim карты - /msisdn – выводит информацию о sim карте по номеру телефона - обработка входящего файла эксель с названием первой колонки ip, ips, iccid, imsi, tel или msisdn ( по ips работает с simcards, по ip c sims) - /meter - команда по добавлению монтажа ПУ - /emeter user@example.com - команда по отправке всех монтажей из БД бота (Замените user@example.com на email куда надо отправить монтаж!!!)
